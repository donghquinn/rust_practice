# 함수

## 함수 선언
- 함수는 fn 을 통해 선언한다.
- 함수의 이름은 snake 표현식으로 표현한다.
    - ex: hello_world
- 함수 문맥의 종료는 {}를 통해 알려준다.

## 표현식과 구문
- 변수에 함수 할당하기

```
    let y = {
        let x = 3
        x + 1
    }
```

## 리턴값
- 리턴 값의 타입은 명시해 줘야 한다.
- 마지막 표현식을 리턴한다. 이 때 리턴되는 표현식에는 세미콜론을 넣으면 안된다.

```
fn calculate (x: i32) -> i32 {
    x + 1
}
```
